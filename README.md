# Chatmeleon

## Setup

Ensure you have python 3.6+ installed.

```bash
pip install -r requirements.txt
or
pip install -r requirements.txt --user
```

## Running the Server

```bash
cd website
python main.py
```

## Clearing Database

To clear the database simply delete the `messages.db` file.
